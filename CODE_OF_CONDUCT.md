# Code of Conduct

This project adopts the
[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- **Be civil.** Treat everyone with respect. Critique ideas, not people.
- **Assume good faith.** Maintainers and contributors give time
  voluntarily; ask before accusing.
- **Stay on topic.** The issue tracker and PRs are for the project, not
  for unrelated discussion.

## Unacceptable behavior

Harassment, sustained personal attacks, doxxing, sexualized content,
threats, or any conduct that creates an unsafe environment for others
is not tolerated and will be moderated without notice.

## Enforcement

Reports go to the maintainers via the email listed in
[SECURITY.md](./SECURITY.md) (`fabrizio.salmi@gmail.com`). Reports are
handled confidentially.

Possible outcomes range from a private warning, to a temporary
suspension from project spaces, to a permanent ban — proportional to
the severity and pattern of the behavior.

## Scope

Applies to all project spaces: this repository, the issue tracker, PR
discussions, and any directly affiliated channels.
